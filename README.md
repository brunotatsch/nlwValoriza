- Cadastro de usuários

  [X] Não é permitido cadastrar usuário sem e-mail

- Cadastro de TAG

  [X] Não é permitido cadastrar mais de uma tag com o mesmo nome

  [X] Não é permitido um usuário cadastrar um elogio para si mesmo
